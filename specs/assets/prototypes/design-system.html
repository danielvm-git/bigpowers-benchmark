SwiftUI Architecture Notes

App Shell: App protocol + Scene + WindowGroup per top-level destination. MenuBarExtra for menu bar item with live run progress.

Navigation: NavigationSplitView (sidebar · content · detail). Collapsible sidebar with @SceneStorage for persistent selection. SF Symbols on every sidebar Label.

Toolbar: .toolbar(content:) + ToolbarItemGroup(placement: .principal / .primaryAction / .navigation). Status pulse in .principal, primary actions in .primaryAction.

Data Tables: Table + TableColumn for sortable, virtualized rows (Tasks Table, Run Explorer, Model Health). TableRow selection for compare-mode multi-select.

Charts: Swift Charts — Chart + LineMark (score evolution), AreaMark (fill under), RuleMark (gridlines), PointMark (data points), BarMark (heatmap → RectangleMark). Skill radar via Canvas + Path + polar coordinate helper.

Forms: Form + Section + LabeledContent + Toggle + Picker(.menu) + SecureField. Settings master-detail via .inspector placement.

Sheets & Inspectors: .sheet(isPresented:) for modals; .inspector(isPresented:) for slide-in detail panels (Registry model inspector).

Terminal Panel: NSViewRepresentable wrapping NSTextView + NSAttributedString from ANSI parser. Blinking caret via Timer + accessibilityReduceMotion. Autoscroll via scrollRangeToVisible.

Run Streaming: Process() + Pipe + AsyncStream<LogLine> consumed by @Observable view model. bench:progress / bench:result map to AsyncStreams on BenchmarkRunner actor.

Persistence: File-backed runs/data/ for GitOps. FileManager + JSONEncoder for BenchRow. DispatchSource.makeFileSystemObjectSource watches directory for external git pulls.

Git Operations: Process() shell-out to system git for commit/push. Avoid bundling libgit2 in v1.

Keychain: Security framework — SecItemAdd / SecItemCopyMatching for API keys keyed by provider ID.

Notifications: UserNotifications (UNUserNotificationCenter) for run-complete / failed / regression. UNNotificationAction for actionable buttons.

Command Palette: Custom Window(_:id:) + .windowStyle(.hiddenTitleBar) + .searchable. Global hotkey via NSEvent.addGlobalMonitorForEvents or HotKey SPM.

Multi-Window: Each WindowGroup gets @SceneStorage view model; shared data via app-level @Observable store + .environment.

Sleep Prevention: IOPMAssertionCreateWithName(kIOPMAssertionTypePreventUserIdleSystemSleep) while run active; release on completion.

Light/Dark: Color extensions with init(light:dark:). Default .colorScheme is .dark (canonical); .preferredColorScheme follows user pick.

Accessibility: @Environment(\.accessibilityReduceMotion) gates animations. VoiceOver labels on charts with data-table fallback. Keyboard nav + visible focus rings.
