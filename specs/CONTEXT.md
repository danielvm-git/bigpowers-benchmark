## OpenCode — The Benchmark Runner

Source: `anomalyco/opencode` (opensrc, `dev` branch).

OpenCode is the AI coding agent being benchmarked. The macOS app launches it as a subprocess.

| Fact | Detail |
|------|--------|
| Install | `brew install anomalyco/tap/opencode` |
| Invocation | `opencode run --model <id> --ref <bigpowers-ref>` |
| Output | stdout/stderr streamed line by line |
| Architecture | TypeScript monorepo (Effect.js + Drizzle + SQLite) |
| Sessions | Each run = a session with parts/messages |
| Desktop app | Available as Electron `.dmg` — separate from the CLI |

**Swift integration pattern** (from `SWIFTUI_HANDOFF.md`):
```swift
actor BenchmarkRunner {
    func runBenchmark(model: Model, ref: String) -> AsyncStream<LogLine>
    // spawns: Process() → /usr/local/bin/opencode run --model id --ref ref
    // pipes stdout+stderr → AsyncStream<LogLine>
}
```

## Data Model (from SWIFTUI_HANDOFF.md)

```swift
struct BenchRow: Codable, Identifiable {
    let id: UUID
    let timestamp: Date
    let bigpowersRef: String    // JSON: "bigpowers_ref"
    let modelName: String
    let taskId: String
    let codePass: Int           // JSON: "code_pass"
    let artifactScore: Int      // JSON: "artifact_score"
    let conventionScore: Int    // JSON: "convention_score"
    let overallScore: Double    // JSON: "overall_score"
    let duration: TimeInterval
    let cost: Double
    let workspace: String
}
// overallScore = (codePass * 2 + artifactScore + conventionScore) / 4
```

JSON shards stored at `~/runs/data/run_<ISO8601>_<taskId>.json`.
